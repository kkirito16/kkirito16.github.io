рҹ’ЎеүҚз«ҜжҸҗж•ҲпҪңд»Җд№ҲжҳҜMonorepoпјҹ

жңҖиҝ‘еңЁеҒҡж–°йЎ№зӣ®ж—¶еҶҚдёҖж¬ЎйҮҮз”ЁдәҶMonorepoпјӣдәҺжҳҜеҖҹжӯӨжңәдјҡдҫҝз»ҷз»„е‘ҳеҲҶдә«дәҶдёҖдёӢд»Җд№ҲжҳҜMonorepo

з”ЁдёҖеҸҘиҜқжҖ»з»“е°ұжҳҜпјҡMonorepo = дёҖдёӘд»“еә“пјҢиЈ…дёӢеӨҡдёӘзӢ¬з«ӢйЎ№зӣ®гҖӮ

рҹ”Қ дёҖеҸҘиҜқи§ЈйҮҠ

MonorepoпјҲMonolithic RepositoryпјүзӣҙиҜ‘еҸ«вҖңеҚ•дҪ“д»“еә“вҖқпјҢ

е®ғи®©еӨҡдёӘйЎ№зӣ®пјҲжҜ”еҰӮдёҖдёӘ React дё»еә”з”Ё + Vue еҗҺеҸ° + UI з»„д»¶еә“ + е·Ҙе…·еә“пјүе…ұеӯҳеңЁеҗҢдёҖдёӘ Git д»“еә“дёӯпјҢ

дҪҶжҜҸдёӘйЎ№зӣ®йғҪиғҪзӢ¬з«ӢејҖеҸ‘гҖҒзӢ¬з«ӢйғЁзҪІгҖҒдә’дёҚе№Іжү°

д№ҹе°ұжҳҜиҜҙпјҡ

е®ғдёҚжҳҜвҖңжҠҠжүҖжңүд»Јз ҒжҸүжҲҗдёҖдёӘвҖқпјҢиҖҢжҳҜвҖңз»ҹдёҖз®ЎзҗҶгҖҒзҒөжҙ»ејҖеҸ‘вҖқгҖӮ

е’ҢвҖңеҚ•дҪ“еә”з”ЁпјҲMonolithпјүвҖқе®Ңе…ЁдёҚжҳҜдёҖеӣһдәӢпјҒ

рҹҡҖ дёәд»Җд№Ҳи¶ҠжқҘи¶ҠеӨҡеүҚз«ҜеӣўйҳҹйҖүжӢ© Monorepoпјҹ

1. д»Јз Ғе…ұдә«ж— ж•Ңдёқж»‘

з»„д»¶еә“гҖҒе·Ҙе…·еҮҪж•°гҖҒTS зұ»еһӢвҖҰвҖҰ

дёҚйңҖиҰҒеҸ‘ npm еҢ…пјҢзӣҙжҺҘжң¬ең°еј•е…ҘпјҢдҝ®ж”№еҚіз”ҹж•ҲпјҢејҖеҸ‘дҪ“йӘҢжһҒдҪі

2. дҫқиө–з®ЎзҗҶжӣҙе№ІеҮҖ

дёҖдёӘд»“еә“дёҖдёӘ node_modulesгҖӮ

з”Ё pnpm з®ЎзҗҶ workspaceпјҢеҪ»еә•е‘ҠеҲ«вҖңдҫқиө–ең°зӢұвҖқгҖӮ

3. дёҖж¬ЎжҸҗдәӨжҗһе®ҡеӨҡдёӘйЎ№зӣ®

йңҖиҰҒеҗҢж—¶ж”№еә”з”Ёе’Ңз»„д»¶еә“пјҹ

дёҖж¬Ў commitпјҢе…ЁйғҪжҗһе®ҡгҖӮиҝҪжәҜеҺҶеҸІжӣҙжё…жҷ°пјҒ

4. з»ҹдёҖе·Ҙе…·й“ҫ

ESLintгҖҒPrettierгҖҒTypeScriptгҖҒJestвҖҰвҖҰ

ж №зӣ®еҪ•й…ҚзҪ®дёҖж¬ЎпјҢе…Ёд»“еә“е…ұдә«и§„еҲҷпјҢеӣўйҳҹеҚҸдҪңжӣҙйЎәз•…гҖӮ

5. еӣўйҳҹеҚҸдҪңжӣҙйҖҸжҳҺ

жүҖжңүйЎ№зӣ®йғҪеңЁдёҖдёӘд»“еә“пјҢжғізңӢеҲ«зҡ„жЁЎеқ—е®һзҺ°гҖҒеҒҡи·ЁйЎ№зӣ® reviewпјҹso easyпҪһ

дҪҶе®ғд№ҹжңүвҖңеқ‘вҖқрҹ‘Ү

вҖўе·Ҙе…·й“ҫй…ҚзҪ®зЁҚеҫ®еӨҚжқӮпјҢйңҖиҰҒеӯҰд№  Lerna / Nx / Turborepoпјӣ

вҖўд»“еә“еӨӘеӨ§дјҡи®© git ж“ҚдҪңеҸҳж…ўпјӣ

вҖўжқғйҷҗжҺ§еҲ¶иҰҒжӣҙз»ҶиҮҙпјҢе°Өе…¶еӨ§еӣўйҳҹиҰҒжіЁж„ҸпјҒ

вҡҷпёҸ еёёи§Ғ Monorepo е·Ҙе…·жҺЁиҚҗ

еҢ…з®ЎзҗҶеҷЁпјҲеҝ…йЎ»пјү

вҖўpnpmпјҡMonorepo зҡ„еӨ©йҖүд№ӢеӯҗпјҢйҖҹеәҰеҝ«гҖҒз©әй—ҙеҚ з”Ёе°ҸгҖҒеҺҹз”ҹж”ҜжҢҒ workspaceгҖӮ

вҖўnpm (v7+) / yarn (v2+)пјҡд№ҹеҸҜд»ҘпјҢдҪҶжҖ§иғҪз•ҘйҖҠгҖӮ

д»»еҠЎзј–жҺ’ & жһ„е»әзі»з»ҹпјҲжҺЁиҚҗпјү

вҖў Turborepoпјҡз”ұ Vercel еҮәе“ҒпјҢжҷәиғҪд»»еҠЎи°ғеәҰ + жһ„е»әзј“еӯҳпјҢеҝ«еҫ—зҰ»и°ұгҖӮ
